# What Is This App

This WEB App stores some taxi geolocation data from Sweden, and lets the user query for them within a given timeframe.

# Technologies Used

- NoSQL with MongoDB to store taxi data.
- SQLite to store user info.
- Node.js for backend.
- EJS for frontend.
